# Shared Responsibility Model

Clarify which controls are owned by the template product team and which remain with the client or deployer.

## Template

For each control area capture:

- control area
- vendor responsibility
- customer responsibility
- shared responsibility
- evidence or artifact
- notes

## Suggested Areas

- identity and access configuration
- tenant onboarding and offboarding
- backup and disaster recovery execution
- infrastructure patching
- secret management
- data classification and retention decisions
- incident communication
