# Transfer Assessment

Record cross-border transfer considerations for processors, hosting, support access, and analytics tools.

## Template

For each transfer scenario capture:

- system or vendor
- origin region
- destination region
- data categories involved
- transfer mechanism
- supplementary controls
- whether the vendor requires cleartext access
- legal escalation required
- residual risk
- owner

## Red Flags

- vendor personnel or systems require routine access to data in cleartext
- transfer relies only on contract language without effective technical safeguards
- support workflows allow unrestricted remote access to production data
- analytics or monitoring vendors receive more personal data than strictly necessary

High-risk or unclear cases should be escalated for legal and privacy review rather than treated as resolved by default.
