# Processor RoPA

Maintain a Record of Processing Activities for the organization in its role as processor.

## Template

For each processing activity capture:

- activity name
- controller or customer category
- purpose of processing
- categories of personal data
- categories of data subjects
- categories of recipients
- subprocessors involved
- cross-border transfers involved
- retention reference
- technical and organizational measures reference
- owner

## Notes

This artifact is especially important for enterprise procurement and audit readiness because it makes processor accountability explicit.
