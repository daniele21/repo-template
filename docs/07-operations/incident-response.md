# Incident Response

## Minimum Incident Flow

1. detect and triage
2. contain impact
3. mitigate safely
4. identify root cause
5. restore service
6. document lessons and systemic remediation

## Required Outputs

- timeline
- impact summary
- root cause
- immediate mitigation
- long-term remediation
- monitoring or process gaps

## Privacy and Breach Coordination

If personal data may be involved, also record:

- categories of personal data affected
- impacted controllers, tenants, or user groups
- notification obligations and timing
- controller notification path and owner
- evidence preserved for audit and follow-up review
